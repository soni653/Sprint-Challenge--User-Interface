Question and Answer:

    1. If you were to describe semantic HTML to the next cohort of students, what would you say?
	   Semantic HTML or semantic markup is HTML that introduces meaning to the web page rather than just presentsation.
	

    2. Name two big differences between display: block; and display: inline;.
       inline elements can be nested inside of other elements without disrupting the flow of content.
	   and block elements will break the flow of content on a new line. You will find that block elements are the backbone of most layouts and are vital to building any web page.
	   
    3.What are the 4 areas of the box model?
	  Every box is composed of four parts (or areas), defined by their respective edges: the content edge, padding edge, border edge, and margin edge.
	  The content area, bounded by the content edge, contains the "real" content of the element, such as text, an image, or a video player

    4. While using flexbox, what axis does the following property work on: align-items: center?
	   align-items: center; The flexbox items are aligned at the center of the cross axis. By default, the cross axis is vertical. This means the flexbox items will be centered vertically.

    5. Explain why git is valuable to a team of developers.
	   Git has powerful branching capabilities. To start work, developers have to first create a unique branch.
	   Each branch functions in an isolated environment while changes are carried out in the codebase. This ensures that the master branch always supports production-quality code.
